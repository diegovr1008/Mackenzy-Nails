# Mackenzy-Nails

HTML

<!DOCTYPE html> → Indica que o documento segue o padrão HTML5.
<html lang="pt-br"> → Tag raiz que envolve todo o conteúdo. O atributo lang="pt-br" informa que o idioma principal é português do Brasil.

<head> → Cabeçalho da pagina - Contém informações que não aparecem diretamente na página, mas são essenciais.
<meta charset="UTF-8"> → Define a codificação de caracteres (UTF-8), permitindo acentos e caracteres especiais.
<meta name="viewport"...> → Faz a página se adaptar a diferentes tamanhos de tela (responsividade).
<title> → Nome que aparece na aba do navegador.
<link rel="stylesheet" href="nail.css"> → Importa o arquivo CSS externo para estilizar a página.
<link href="https://fonts.googleapis.com/..."> → Importa uma fonte personalizada do Google Fonts.

<body> Corpp - é o conteúdo visível para o usuário.

<header> → Cabeçalho da página.
<h1> → Título principal.
<nav> → Menu de navegação.
<a href="#..."> → Links internos que levam para seções específicas da página.

<section> → Agrupa conteúdo relacionado em sessões.
id="inicio" → Permite que o menu navegue até essa parte.
<h2> → Subtítulo da seção.
<p> → Parágrafo de texto.
<a href="..."> → Link para abrir o WhatsApp.
target="_blank" → Abre em nova aba.
<button> → Botão estilizado via CSS.

<ul> → Lista não ordenada.
<li> → Cada item da lista.

<div> quando quiser agrupar elementos sem um significado específico.
<div> significa division (divisão). Serve principalmente para organização e aplicação de estilos via CSS ou manipulação com JavaScript.

class =  é usado para nomear um elemento ou grupo de elementos. Você pode usar a mesma classe em vários elementos, e todos receberão o mesmo estilo.
